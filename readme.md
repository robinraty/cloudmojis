# Cloudmoji ☁️

An interactive cloud-themed emoji clicker game built with HTML, CSS and JavaScript.

## Objective
You have 10 seconds to score as many points as possible by clicking positive emojis while avoiding dangerous ones.

## Rules
- ⭐ Star = +1 point
- 💎 Diamond = +3 points (rare)
- ☄️ Meteorite = -1 point
- 💣 Bomb = Instant game over

## Features
- Fast-paced gameplay
- Dynamic scoring system
- Random emoji spawning
- Visual feedback
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
